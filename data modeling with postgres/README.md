# Data Modeling with Postgres

## Overview
In this project, I apply data modeling with postgres and build a ETL pipeline using python.

## Technical Aspect
I've used python to connect to postgresql and execute some queries. I've created the database, and inserted data.

## Installation
The code is written in Python 3.8. If you don't have Python installed, you can find it here(https://www.python.org/downloads/).
